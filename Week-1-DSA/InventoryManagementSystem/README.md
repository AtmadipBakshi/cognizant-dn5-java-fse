# Inventory Management System

## Objective
To manage warehouse inventory using efficient data structures.

---

## Data Structure Used
We use **HashMap**:
- Key: productId
- Value: Product object

---

## Why HashMap?
- Fast access using key
- Efficient add, update, delete operations

---

## Operations & Time Complexity

### Add Product
- Time: O(1)

### Update Product
- Time: O(1)

### Delete Product
- Time: O(1)

### Display
- Time: O(n)

---

## Conclusion
HashMap is the most efficient choice for inventory systems due to constant time operations for insert, update, and delete.
